Install as App
🍎 iPhone & iPad (Safari)
01
Open this page in Safari — it must be Safari, not Chrome or Firefox.
02
Tap the Share button — the box with an arrow pointing up, at the bottom of the screen.
03
Scroll down the share sheet and tap Add to Home Screen.
04
Confirm the name and tap Add.
The app icon will appear on your home screen and launch full-screen.
05
First launch: when prompted for location, choose Allow While Using App.
Settings → Privacy → Location Services → Safari if you need to change it later.
🤖 Android (Chrome)
01
Open this page in Chrome — not Samsung Browser or Firefox.
02
Look for an Install app banner at the bottom of the screen and tap it.
If no banner appears, continue to the next step.
03
If no banner, tap the ⋮ menu in the top-right corner, then tap Add to Home Screen.
04
Tap Add to confirm. The app will appear in your app drawer and on your home screen.
05
First launch: allow location access when Chrome asks.
If the install option doesn't appear, reload the page and wait a few seconds for the app to fully initialize, then try again.
Once installed, Field Locator works offline for all features except state & county lookup, which requires a network connection.
